# Side App

웹 애플리케이션 프로젝트

## 프로젝트 구조

```
side-app-done/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css       # 스타일시트
├── js/
│   ├── app.js          # 메인 애플리케이션 로직
│   └── storage.js      # 로컬 스토리지 관리 유틸리티
├── images/             # 이미지 파일 디렉토리
└── README.md           # 프로젝트 설명
```

## 시작하기

1. `index.html` 파일을 브라우저에서 열어 실행합니다.

## 기능

- 로컬 스토리지 관리 기능 제공
